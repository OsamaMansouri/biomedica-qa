# Gherkin acceptance criteria

`.feature` files here document **acceptance criteria** for smoke and e2e flows. They are **not executed** in CI.

**Execution:** Playwright specs in [`../tests/`](../tests/).

**Traceability:** `Gherkin_Feature` column in [`../../docs/spreadsheets/smoke-catalog.csv`](../../docs/spreadsheets/smoke-catalog.csv) and [`e2e-catalog.csv`](../../docs/spreadsheets/e2e-catalog.csv).

Tags (`@cart`, `@smoke`, `@e2e`, …) align with tags in test titles.
